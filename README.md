# Advanced Calculator
A Java-Base **Advanced Calculator Web Application** calculator that evaluate complex mathematical expressions with multiple operations in a single input String.

Built using **HTML, CSS, JavaScript, Java, Spring Boot, and Thymeleaf**.  
This calculator supports **multiple operations in a single expression**, provides an **interactive UI**, and handles different types of calculation errors efficiently.

## 🚀 Features

- Perform multiple operations in a single expression
- Real-time calculation as operands are added
- Supports arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  - Modulus (%)
- Interactive and responsive user interface
- Backspace button to remove the last entered value
- Error handling for invalid expressions
- Prevents invalid inputs
- Smooth user experience

## ⚙️ How It Works

1. The user enters numbers and operators through the UI.
2. As a new operand is entered, the calculator evaluates the previous operation.
3. The expression is processed using Java backend logic.
4. Results are displayed dynamically on the interface.
5. Errors such as invalid input or division by zero are handled gracefully.
